# SimpleDEX
Front-End Proyecto SimpleDEX


Se sumo una version de SimpleDEX considerando las sugerencias de las correcciones.
Se busco optimizar la eficiencia del uso del gas:
Corrigiendo la indexacion innesaria en las llamadas a eventos.
Haciendo cambios en el uso del if en la funcion "getprice"
Se eliminaron variables de estado para actualizacion del estado del pool, reemplazando por variables locales segun el caso.
